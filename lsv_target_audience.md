# Анализ целевой аудитории страховой компании
### Описание проекта: 
Цель проекта: создание дашборда для страховой компании для анализа целевой аудитории

Используемые технологии: PostgreSQL, Yandex Datalens

Цель дашборда: Изучение аудитории компании и приносимлй прибыли от разных категорий пользователей.

Заказчик: Отдел продаж.

Ссылка на дашборд: https://datalens.ru/qiegbetvdpaab
